This module runs an extensive test on the current page where all possible CSS properties are animated randomly and for each animation the code in the test tries to detect layout changes by comparing getBoundingClientRects() from all elements. Note that the test runs for elements under the tree of the selected element ($0).

Parameter Value
PRNG Seed
Share of Elements to Animate (%) (roughly on page)
Measurement Period (raf)
Test Animation Duration (ms)
Number of Runs Per (Animation, Element)
Custom CSS Properties to Test (.json list)
